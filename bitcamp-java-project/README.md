## src03 - 클래스와 배열
 - 학습목표
   - 클래스를 사용하여 입력 데이터를 묶어서 저장하는 방법을 익힌다.
   - 배열을 사용하여 여러 개의 데이터를 다루는 방법을 익힌다.
   - for 반복문을 사용하여 배열을 다루는 방법을 이해한다.
   - if 조건문을 사용하는 방법을 이해한다.
   - 문자열을 비교하는 방법을 이해한다.
 - 작업내용
   - 사용자로부터 팀정보를 입력 받아 출력하라.
   - 입력은 최대 5개까지 가능하다.
   - 5개를 넘으면 바로 출력한다.
 - 실행결과
